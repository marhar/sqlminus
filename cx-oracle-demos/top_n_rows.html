top N rows of query

This select the first N rows of a query. If you are looking for the SELECT...LIMIT clause this is it.

Note that ROWNUM starts at 1 and not 0.

# the first 5 rows of the subquery
curs.execute("""select * from (
                  select a,b from cxdemo_t1 order by a
                ) where rownum <= 5""")
for r in curs:
    print r[0]
