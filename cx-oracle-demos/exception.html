Handling an oracle exception

cx_Oracle will throw cx_Oracle.DatabaseError exceptions. Here's how to extract the oracle error code and error message.

try:
    curs.execute('select 1/0 from dual')
except cx_Oracle.DatabaseError, e:
    print "Exception:", e
    errorObj, = e.args
    print "   ErrorCode:", errorObj.code
    print "ErrorMessage:", errorObj.message
